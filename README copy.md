## 项目结构

- `src/`：主程序代码
- `scripts/`：辅助脚本
- `data/raw/`：原始数据
- `data/processed/`：处理后的数据
- `logs/`：日志文件
- `tests/`：测试代码

## 常用命令

### 创建虚拟环境

    python -m venv .venv

### 激活虚拟环境

    source .venv/bin/activate

### 安装依赖

    pip install -r requirements.txt

### 运行主程序

    python src/main.py
